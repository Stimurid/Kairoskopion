envelope, which source layer is the primary computation, which are
corroborating, and which are weak or forbidden?**

If this file disagrees with the canon, the canon wins.

## 3. Six fundamental caveats

These are the non-negotiable interpretation rules for the rubric.
They apply at every funnel layer. The scorer and reviewer hard-fail
on violation.

### 3.1 Official scope = vendor / official claim, not fit fact

A journal's `aims & scope` text is the venue's self-description. It
is `official_claim` authority at best. It does **not** prove the
venue actually publishes work in those areas. Use D corpus distribution
to verify. A `discipline_envelope` derived only from A is `INFERENCE`
with confidence `low`.

### 3.2 Indexing = strategic / publication-regime signal, not fit

Scopus / WoS / DOAJ inclusion (and quartile snapshots) populate
`institutional_signals` and inform `publication_regime_fit` and
`trust_compliance_fit`. They do **not** populate `discipline_fit`,
`school_fit`, `argument_move_fit`, or any other content-fit axis.
"Q1 in philosophy" does not make a venue a fit for a continental
essay; it only means the regime is prestigious.

### 3.3 Full text = corpus material, not metadata authority

The body of an article (PDF, HTML, OA repository copy, Sci-Hub-like
mirror, ResearchGate upload, personal library, user-uploaded ZIP)
is **the material for corpus mining**: method classification,
references extraction, language register analysis. It is **never** the
source of truth for the venue's identity, policies, indexing, or
editorial board. Those must come from A / B / C / E / G separately.

### 3.4 Editorial board = cloud / inference, not psychology

`EditorialBoardCloud` is a **distribution** over institutions,
countries, disciplines, theoretical traditions. It is **not** a
psychological profile of individual editors. Derived signals
("disciplinary center of gravity", "geographic center", "methodological
center") are `INFERENCE` and carry confidence flags. Strong statements
about individuals ŌĆö "Editor X dislikes continental theory" ŌĆö are
forbidden. Allowed: "available board publication record shows Ōēż 1
continental contribution out of N sampled; confidence low; sample
bias possible."

### 3.5 Unknown ŌēĀ absent

If the rubric cannot establish a field, the field is marked
`UNKNOWN_NOT_VERIFIED` with the reason in `unknowns`. It is **not**
filled with a default, fabricated, or null-coerced value. A missing
DOAJ entry does not mean "no OA"; it means "not verified". A missing
editorial board page does not mean "no board"; it means "not parsed".

### 3.6 Incomplete graph ŌēĀ absence of tradition

If `EditorialBoardCloud` samples 6 of 25 editors and finds no
Simondonians, the rubric must NOT conclude "this venue is unfriendly
to Simondon". It must conclude "sample insufficient; tradition
representation undetermined". Same for citation graph: 30 articles
without a name does not mean the name is taboo there; it means we
need more sample.

## 4. Cache-miss authority map

Aligned with canon ┬¦7 four-state taxonomy (`absent` / `stale` /
`weak_evidence` / `fresh_sufficient`). For each state, which adapter
calls run and what they cost. **No LLM calls** for any state ŌĆö adapters
fetch, registry stores, LLM interprets at fit time (Agentum policy).

| Cache-miss state | What we fetch | Source categories | Cost |
|---|---|---|---|
| `absent` (no `VenueRecord`) | identity ŌåÆ light envelope from A+B+C + 5ŌĆō10 D sample | A + B + C + G + small D | medium (5ŌĆō10 HTTP, no LLM) |
| `stale` on `FormalSubmissionProfile` | re-fetch A + B | A + B | low |
| `stale` on `PublishedCorpusHull` | refresh last N articles | G + small D | medium |
| `weak_evidence` on `EditorialBoardCloud` | sample more editors + ORCID + OpenAlex | E + G | medium-high |
| `weak_evidence` on `CitationExpectationProfile` | enlarge corpus refs to 50 + OpenCitations | D + G | high |
| `weak_evidence` on `geographic_affinity` envelope | aggregate F + E | E + F | medium |
| `fresh_sufficient` | nothing ŌĆö compose from registry | none | zero |

Adapter mode (`fixture` / `live` / `cached`) is **orthogonal** to
cache-miss state: state decides what to ask for; mode decides what
gets returned. An `absent` state with `fixture` mode produces
fixture data; with `live` mode it makes the HTTP call.

## 5. Forbidden source uses (hard FAIL gates)

For the scorer and reviewer.

- `discipline_envelope` populated only from A scope ŌĆö forbidden;
  must include D corroboration, else `partial`.
- `institutional_signals.indexing` populated only from A self-claim ŌĆö
  forbidden; must include C registry corroboration, else
  `UNKNOWN_NOT_VERIFIED`.
- `EditorialBoardCloud` derived signal from < 4 editors of a board
  Ōēź 10 ŌĆö forbidden; must be `partial` with explicit sample size.
- Any subobject populated from H without independent A / B / C / G
  metadata ŌĆö forbidden. H is body, not metadata.
- Any `tacit_venue_signal` without source-and-date stamp ŌĆö drops
  `evidence_status` to `UNKNOWN_NOT_VERIFIED`.
- Any envelope range with `lo > hi` ŌĆö schema-invariant violation,
  rejected.
- Any `school_envelope` listing a tradition the corpus did not exhibit
  ŌĆö forbidden as fabrication; mark as `absent_but_relevant` instead.

## 7. How the rubric is consumed

- **Scorer** (`scripts/run_venue_side_benchmark.py`): ┬¦3 caveats and ┬¦5
  hard FAIL gates become per-axis assertions. ┬¦6 ranges become
  PASS/PARTIAL thresholds for shape match.
- **Operator review:** for any venue profile with `INFERENCE` status
  on an axis whose primary layer per ┬¦2 is D / E / G, the reviewer
  must check the run actually hit D / E / G or only A. If only A, the
  axis is downgraded to `partial`.
- **Adapter authors:** new adapters declare which source category
  (AŌĆōJ) they belong to. The authority is enforced per ┬¦2.
